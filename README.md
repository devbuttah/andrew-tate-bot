SETUP:
1. Clone the repository (git clone https://github.com/devbuttah/andrew-tate-bot.git)
2. Navigate to the project directory
3. Make sure to have node and the required dependencies installed
4. Run "node deploy-commands.js" to register the commands.
5. Run "node index.js" to start the bot.

IMPORTANT!

Make sure to create a config.json file and put in:

{
	"token": "YOURTOKEN",
  "clientId": "YOURCLIENTID",
  "guildId": "YOURGUILDID"
}
